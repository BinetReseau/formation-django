d. Client & serveur

  • Deux parties découplées d'un même projet
  • Permet d'être modulaire
  • Ex : application Deezer pour iPhone :
    • l'application sur le téléphone est le client
    • elle effectue des requêtes à un serveur de Deezer
    • le serveur répond et envoie les données au client
  • Ex : Chocapix :
    • En tapant http://chocapix/ dans la barre d'adresse, on récupère des fichiers HTML et JavaScript qui sont exécutés par le navigateur :
      c'est le client, qui est une application exécutée dans le navigateur.
    • Ce client est capable d'effectuer des requêtes au serveur de Chocapix.
